Exercise 2: A Circle class

Create a class circle with radius(double).

Implement:

Default constructor

Parametrized constructor

Copy constructor

Assignment operator

Destructor

operator<< to print "Radius: r".

In main(), create, copy, and print points.

Your main:

int main() {
    Circle c1;
    Circle c2(7.5);
    Circle c3 = c2;
    c1 = c3;

    cout << c1 << endl;
    cout << c2 << endl;
    cout << c3 << endl;

    return 0;
}
