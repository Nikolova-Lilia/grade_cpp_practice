Exercise 20: A Shoe class

Create a class Shoe with size (int) and color (string).

Implement:

Default constructor

Parametrized constructor

Copy constructor

Assignment operator

Destructor

operator<< to print "Size: s, Color: c".

In main(), create, copy, and print points.

Your main:

int main() {
    Shoe s1;
    Shoe s2(42, "Black");
    Shoe s3 = s2;
    s1 = s3;

    cout << s1 << endl;
    cout << s2 << endl;
    cout << s3 << endl;
    return 0;
}
