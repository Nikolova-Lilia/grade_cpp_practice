Exercise 8: A Laptop class

Create a class Laptop with brand (string) and ram (int, GB).

Implement:

Default constructor

Parametrized constructor

Copy constructor

Assignment operator

Destructor

operator<< to print "Brand: b, RAM: r GB".

In main(), create, copy, and print points.

Your main:

int main() {
    Laptop l1;
    Laptop l2("Dell", 16);
    Laptop l3 = l2;
    l1 = l3;

    cout << l1 << endl;
    cout << l2 << endl;
    cout << l3 << endl;

    return 0;
}
