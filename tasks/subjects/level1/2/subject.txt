Exercise 2: A Rectangle class

Create a class rectangle with width and height (ints): w,h.

Implement:

Default constructor (0,0)

Parametrized constructor (w,h)

Copy constructor

Assignment operator

Destructor

operator<< to print "Width: w, Height: h".

In main(), create, copy, and print points.

Your main:

int main() {
    Rectangle r1;
    Rectangle r2(5, 10);
    Rectangle r3 = r2;
    r1 = r3;

    cout << r1 << endl;
    cout << r2 << endl;
    cout << r3 << endl;

    return 0;
}
