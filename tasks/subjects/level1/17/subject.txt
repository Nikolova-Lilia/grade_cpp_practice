Exercise 17: A Bicycle class

Create a class Bicycle with type (string) and gears (int).

Implement:

Default constructor

Parametrized constructor

Copy constructor

Assignment operator

Destructor

operator<< to print "Type: t, Gears: g".

In main(), create, copy, and print points.

Your main:

int main() {
    Bicycle b1;
    Bicycle b2("Mountain", 21);
    Bicycle b3 = b2;
    b1 = b3;

    cout << b1 << endl;
    cout << b2 << endl;
    cout << b3 << endl;
    return 0;
}
