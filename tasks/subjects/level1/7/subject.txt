Exercise 7: An Employee class

Create a class Employee with name (string) and salary (double).

Implement:

Default constructor

Parametrized constructor

Copy constructor

Assignment operator

Destructor

operator<< to print "Name: n, Salary: s".

In main(), create, copy, and print points.

Your main:

int main() {
    Employee e1;
    Employee e2("Bob", 45000.5);
    Employee e3 = e2;
    e1 = e3;

    cout << e1 << endl;

    cout << e3 << endl;

    return 0;
}
