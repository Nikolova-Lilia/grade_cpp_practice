Exercise 15: A Chair class

Create a class Chair with material (string) and weight (int, kg).

Implement:

Default constructor

Parametrized constructor

Copy constructor

Assignment operator

Destructor

operator<< to print "Material: m, Weight: w kg".

In main(), create, copy, and print points.

Your main:

int main() {
    Chair c1;
    Chair c2("Wood", 12);
    Chair c3 = c2;
    c1 = c3;

    cout << c1 << endl;
    cout << c2 << endl;
    cout << c3 << endl;
    return 0;
}
