Exercise 13: A TV class

Create a class Smartphone with model (string) and battery (int, mAh).

Implement:

Default constructor

Parametrized constructor

Copy constructor

Assignment operator

Destructor

operator<< to print "Model: m, Battery: b mAh".

In main(), create, copy, and print points.

Your main:

int main() {
    Smartphone s1;
    Smartphone s2("iPhone 15", 3200);
    Smartphone s3 = s2;
    s1 = s3;

    cout << s1 << endl;
    cout << s2 << endl;
    cout << s3 << endl;
    return 0;
}
